# Vision-Based Real-Time Drowsiness Detection System

Driver drowsiness detection system using Python, OpenCV and dlib to detect driver fatigue and trigger an alarm for safety.

---

## 📌 Overview
Drowsiness detection is a safety technology that helps prevent accidents caused by drivers falling asleep while driving.  
This system uses computer vision techniques to monitor eye movements and detect signs of fatigue.

---

## 🛠 Technologies Used
- Python
- OpenCV
- dlib
- NumPy
- Haar Cascade Classifier

---

## 🚀 Features
- Real-time face detection
- Eye aspect ratio calculation
- Driver fatigue detection
- Alarm alert when drowsiness is detected

---

## 📂 Project Files
- `drowsiness_yawn.py` → Main detection code
- `shape_predictor_68_face_landmarks.dat` → Facial landmark model
- `haarcascade_frontalface_default.xml` → Face detection model
- `Alert.wav` → Alarm sound
- `requirements.txt` → Required Python libraries

- ## Download Required Model

Download the dlib facial landmark model from:

http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

Extract the file and place it in the project folder before running the program.
